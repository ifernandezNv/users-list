# Users List Project

## Used Technologies
Project created with the following technologies:
- ReactJS.
- TypeScript.
- ReactRouterDom v6.23.1
- Redux v9.1.2
- Redux toolkit
- Axios.
- TailwindCss.
- Npm

## How to run this project?
In order to use, do as follows:
- Run the comand npm i to install all dependencies.
- Execute npm run dev.
- That's it.
